## REST API
LIst of basic routes:

Route | HTTP | Description
---|---|---
/api/hello?name={name}|GET|Print hello, {name}!

List of user routes:
Route|HTTP|Description
/api/users|GET|Get all the users
/api/users/:id|GET|Get a single user
/api/users|POST|Create a user
